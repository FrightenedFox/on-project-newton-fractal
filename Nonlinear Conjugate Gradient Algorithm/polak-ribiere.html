Metoda gradientów sprzężonych Polaka-Ribiere'a¶

Projekt nr 2 z przedmiotu Optymalizacja Nieliniowa. Autor: Vitalii Morskyi

Podstawa teoretyczna¶

Metoda gradientw sprzonych:

$$ d^{(0)} = -\nabla f(x^{(0)}) $$$$ d^{(i+1)} = -\nabla f(x^{(i+1)}) + \beta^{(i+1)}d^{(i)} $$

gdzie w metodzie Polaka-Ribiere'a: $$ \beta^{(i+1)} = \frac{\nabla f(x^{(i+1)})^T \left( \nabla f(x^{(i+1)}) - \nabla f(x^{(i)}) \right)}{\nabla f(x^{(i)})^T\nabla f(x^{(i)})} $$

Algorytm obliczania wartoci minimalnej podany jest w nastpujcy sposb:

(rdo)

Zadanie projektowe¶

Firma produkuje pomidory i papryk i sprzedaje je po cenie 5 z za kilogram pomidorw i 8 z za kilogram papryki. Koszt produkcji $x$ kg pomidorw i $y$ kg papryki wynosi $C(x, y) = x^2 + 0.9xy\sin(x)\sin(y) + y^2$. Przychd ze sprzeday $x$ kg pomidorw i $y$ kg papryki wynosi $R(x, y) = 5x + 8y$, a zysk to $P(x, y) = R(x, y) C(x, y)$. Ile pomidorw i papryki powinna wyprodukowa firma, aby uzyska maksymalny zysk? (rdo)

Rozwiązanie¶

Funkcja, ktr trzeba zmaksymalizowa:

$$ P(x,y) = R(x,y)-C(x,y) = 5x+8y - x^2 - 0.9xy\sin(x)\sin(y) - y^2 $$

Moemy zapisa j ze znakiem przeciwnym, wtedy otrzymamy problem minimalizacji:

$$ f(x,y) = x^2 + 0.9xy\sin(x)\sin(y) + y^2 - 5x - 8y $$

def f(x):
    """Funkcja celu"""
    # return x[0] ** 2 + 3 * x[0] * x[1] + x[1] ** 2 - 2 * x[0] - 3 * x[1]
    return x[0] ** 2 + 0.9 * anp.sin(x[0]) * anp.sin(x[1]) * x[0] * x[1] + x[1] ** 2 - 5 * x[0] - 8 * x[1]

def polak_ribiere(func, grad_func, Xj, alpha_1=1e-4, alpha_2=2e-1, tol=1e-6):
    """Nieznacznie zmieniona wersja rozwizania rozwizania z tej strony: 
    https://indrag49.github.io/Numerical-Optimization/conjugate-gradient-methods-1.html
    """
    D = grad_func(Xj)
    delta = -D
    iteration = 0
    all_xs, all_ys, all_fs = [Xj[0]], [Xj[1]], [f(Xj)]
    while True:
        print(
            f"It {iteration}\t{Xj=}\t{f(Xj)=:.5f}\tgrad={np.linalg.norm(grad_func(Xj)):.5f}")
        beta = line_search(
            f=func,
            myfprime=grad_func,
            xk=Xj,
            pk=delta,
            c1=alpha_1,
            c2=alpha_2
        )[0]
        if beta is not None:
            X = Xj + beta * delta

        if np.linalg.norm(grad_func(X)) < tol:
            return X, func(X), all_xs, all_ys, all_fs
        else:
            Xj = X
            d = D
            D = grad_func(Xj)
            chi = (D-d).dot(D)/(np.linalg.norm(d) ** 2)
            chi = max(0, chi)
            delta = -D + chi * delta
            
        all_xs += [Xj[0], ]
        all_ys += [Xj[1], ]
        all_fs += [f(Xj), ]
        iteration += 1

def plot_sol(x_path, y_path, f_path):
    meshgrid = xs, ys = np.mgrid[-5:15:0.1, -5:15:0.1]
    z_data = np.array(list(map(f, meshgrid.T.reshape(40000, 2)))
                    ).reshape(200, 200).T
    fig = go.Figure(data=[go.Surface(x=xs, y=ys, z=z_data)])
    fig.update_traces(contours_z=dict(show=True, usecolormap=True,
                                    highlightcolor="limegreen", project_z=True))
    fig.add_trace(go.Scatter3d(
            x=x_path, y=y_path, z=f_path,
            marker=dict(
                size=3,
            ),
            line=dict(
                color='white',
                width=2
            )
        ))
    fig.update_layout(title='Wykres funkcji celu', autosize=True,
                    scene_camera_eye=dict(x=1.87, y=0.88, z=-0.64),
                    # width=1000, height=1000,
                    margin=dict(l=65, r=50, b=65, t=90)
                    )

    fig.show()
